# Bank-Loan-Project
# 📊 Bank Loan Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an **interactive Power BI dashboard** for analyzing bank loan data.
It provides insights into loan applications, funded amounts, repayments, and risk segmentation (Good vs Bad loans).

The dashboard is divided into **three main sections**:

* Summary
* Overview
* Details

---

## 📷 Dashboard Preview

### 🔹 Summary Page

* Total Loan Applications: **38.6K**
* Total Funded Amount: **$435.8M**
* Total Amount Received: **$473.1M**
* Good Loan Issued: **86%**
* Bad Loan Issued: **13.8%**

Key insights:

* Majority of loans are **good loans**
* High repayment performance observed

---

### 🔹 Overview Page

Provides visual insights such as:

* 📈 Monthly funded loan trends
* 🌍 State-wise loan distribution (Map)
* ⏳ Loan term analysis (36 vs 60 months)
* 👨‍💼 Employee length impact
* 💳 Loan purpose breakdown
* 🏠 Home ownership analysis

---

### 🔹 Details Page

* Complete transactional dataset
* Includes:

  * Loan ID
  * Purpose
  * Home Ownership
  * Grade & Sub-grade
  * Issue Date
  * Funded Amount
  * Interest Rate
  * Installment
  * Amount Received
